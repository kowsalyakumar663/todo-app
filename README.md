# Kowsalya — To-Do List App

Interactive client-side task management app built for Thiranex Internship Assignment 3.

##  Live Demo
https://kowsalyakumar663.github.io/todo-app.html

##  Features
- Full CRUD (Create, Read, Update, Delete)
- Auto-save with localStorage
- Filter by All / Active / Completed
- Progress bar
- Dark / Light mode
- Dynamic DOM with event listeners
